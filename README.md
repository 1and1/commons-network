A library of IP network representation classes, like
* IP (IPv4, IPv6),
* MAC address,
* IP network.

## Prerequisites

* Java 8 or later
* Maven 3.6 or later
 
Usage
============
You can include commons-network as a Maven dependency from [Maven Central](https://mvnrepository.com/artifact/com.oneandone/commons-network).

```xml
<dependency>
    <groupId>com.oneandone</groupId>
    <artifactId>commons-network</artifactId>
    <version>0.9.3</version>
</dependency>
```
The JavaDoc documentation can be found here: [![javadoc](https://javadoc.io/badge2/com.oneandone/commons-network/javadoc.svg)](https://javadoc.io/doc/com.oneandone/commons-network)
